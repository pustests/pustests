Пример приложения с PUSH-уведомлениями

Это пример страницы, на которой можно оформить подписку на PUSH-уведомления

После того как вы нажали на кнопку "Включить ПУШ-уведомления", приложению становятся доступны данные подписки, для отправки уведомлений требуется передать эти данные на сервер.

В реальной жизни это делается с помощью отправки запроса на бэкэнд сайта, но т.к. хостинг github позволяет размещать только статичные файлы, данные подписки необходимо перенести на сервер вручную. Скопируйте появившиеся внизу данные подписки, перейдите по этой ссылке, вставьте скопированный код в поле "JSON подписки", выберите приложение к которому относится подписка (первое в списке), после чего нажите кнопку сохранить.
